# thesis_tabpfn
Code for my Bachelor thesis "Application of TabPFN for Predictive Modeling Using Real-World Data in Lung Cancer Patients"

- TabPFN_apollo.ipynb : main notebook with the code
- config_auc_os6.json and config_auc_os24.json : hyperparameter configurations for OS6 and OS24 respectively
- tuned_tabpfn_custom.py : custom version of Tuned TabPFN to support f1-macro and ROC AUC as optimization metrics
